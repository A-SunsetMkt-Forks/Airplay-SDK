# Airplay Receiver For macOS / Ubuntu

For running on macOS, follow these steps:

```bash
brew install ffmpeg@4 sdl2
brew link ffmpeg@4
brew link sdl2
 
Then run airdemo and connect to "Bluberry macOS" 
```
